# Page des Stocks : Consultation et Gestion des Réchauds

Ce guide explique comment accéder à votre inventaire, comprendre les statuts et effectuer des recherches rapides.

## Accès à la page des Stocks

**Question fréquente :** « Comment voir mon stock ? », « Où est l'inventaire ? », « Comment consulter les réchauds disponibles ? »

**Procédure :**

- Cliquez sur l'icône **Stocks** (ou **Inventaire**) dans la **barre de navigation inférieure**
- La page s'ouvre automatiquement sur l'onglet **Stocks actuels**

## Organisation de la page - Deux onglets principaux

| **Onglet**             | **Contenu**                                |
| ---------------------- | ------------------------------------------ |
| **Stocks actuels**     | Liste complète de tous les **réchauds**    |
| **Numéros de Facture** | Suivi des factures disponibles / utilisées |

### Onglet « Stocks actuels » (Réchauds)

#### Indicateurs visuels des réchauds

| **Apparence**                   | **Statut**      | **Signification**                               |
| ------------------------------- | --------------- | ----------------------------------------------- |
| **Cadre vert**                  | Disponible      | Réchaud prêt à être vendu                       |
| **Cadre rouge + badge "Vendu"** | Vendu / Affecté | Réchaud déjà vendu → plus disponible à la vente |

**Astuce :** Le cadre vert/rouge est le moyen le plus rapide de vérifier la disponibilité.

### Onglet « Numéros de Facture »

| **Apparence**                   | **Statut** | **Signification**                 |
| ------------------------------- | ---------- | --------------------------------- |
| **Cadre vert**                  | Disponible | Facture prête à être utilisée     |
| **Cadre rouge + badge "Vendu"** | Utilisée   | Facture déjà associée à une vente |

## Fonctionnalité - Recherche rapide

Une **barre de recherche** est disponible en haut de la page (dans les deux onglets).

**Exemples de recherche :**

- Par modèle : OLI-c / OLI-60b / Parabole
- Par numéro de série : T123456
- Recherche combinée : OLI-45c T258963

## Réponses recommandées de l'Assistant Lemadio

**Utilisateur :** « Comment voir mon stock ? » ou « Où est l'inventaire ? » **Assistant :** Cliquez sur l'icône **Stocks** en bas de l'écran → vous arrivez directement sur la liste de vos réchauds.

**Utilisateur :** « Comment savoir si un réchaud est disponible ? »

**Assistant :** Regardez la couleur du cadre :

- **Vert** → disponible, vous pouvez le vendre
- **Rouge + badge "Vendu"** → déjà vendu, plus disponible

**Utilisateur :** « Comment chercher un réchaud précis dans le stock ? »

**Assistant :** Sur la page Stocks, tapez dans la barre de recherche en haut :

- Le modèle (ex : OLI-60b)
- Ou le numéro de série (ex : T123456)
- Ou les deux ensembles

**Utilisateur :** « C'est quoi un cadre rouge sur un réchaud ? »

**Assistant :** Cadre rouge + badge "Vendu" = ce réchaud a déjà été vendu, il n'est plus dans votre stock de vente.
