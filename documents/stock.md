# Gestion et Consultation du Stock dans Lemadio

**Mots-clés** : voir stock, consulter stock, inventaire, réchaud disponible, réchaud vendu, recherche réchaud, numéro de série, facture disponible

## Comment accéder à la page Stock ?
1. Clique sur l’icône **Stocks** dans la barre de navigation en bas de l’écran  
2. Tu arrives directement sur l’onglet **Stocks actuels**

## Les 2 onglets principaux

| Onglet               | Contenu                                          |
|----------------------|--------------------------------------------------|
| Stocks actuels       | Liste complète de tous les réchauds physiques   |
| Numéros de Facture   | Factures disponibles ou déjà utilisées           |

## Onglet « Stocks actuels » – Statut des réchauds

| Apparence                    | Statut         | Signification                                          |
|------------------------------|----------------|--------------------------------------------------------|
| Cadre **vert**               | Disponible     | Réchaud prêt à être vendu                              |
| Cadre **rouge** + badge “Vendu” | Vendu / Affecté | Réchaud déjà vendu → plus disponible à la vente       |

**Astuce la plus rapide** : la couleur du cadre = disponibilité immédiate.

## Onglet « Numéros de Facture »

| Apparence                    | Statut         | Signification                                          |
|------------------------------|----------------|--------------------------------------------------------|
| Cadre **vert**               | Disponible     | Facture prête à être utilisée dans une vente          |
| Cadre **rouge** + badge “Vendu” | Utilisée    | Facture déjà associée à une vente                     |

## Recherche rapide dans le stock 

Barre de recherche en haut des deux onglets.

**Exemples qui marchent parfaitement** :
- Modèle seul → `OLI-c` `OLI-60b` `Parabole`
- Numéro de série → `T123456`
- Les deux → `OLI-45c T258963`

## Résumé ultra-rapide 
Pour voir ton stock → icône Stocks en bas → cadre vert = vendable, cadre rouge = déjà vendu.  
Pour chercher un réchaud → tape le modèle ou le numéro de série dans la barre en haut.

## Réponses types prêtes à l’emploi

**Question** : Comment voir mon stock ?  
**Réponse** : Clique sur l’icône **Stocks** en bas → tu vois immédiatement tous tes réchauds.

**Question** : Comment savoir si un réchaud est disponible ?  
**Réponse** : Regarde la couleur du cadre :  
- **Vert** → disponible, tu peux le vendre  
- **Rouge + badge “Vendu”** → déjà vendu, plus disponible

**Question** : Comment chercher un réchaud précis ?  
**Réponse** : Sur la page Stocks, tape dans la barre de recherche en haut :  
- le modèle (ex : OLI-60b)  
- ou le numéro de série (ex : T123456)  
- ou les deux ensemble

**Question** : C’est quoi un cadre rouge sur un réchaud ?  
**Réponse** : Cadre rouge + badge “Vendu” = ce réchaud a déjà été vendu, il n’est plus dans ton stock de vente.
